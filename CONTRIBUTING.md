# Contributing to bark-web

# Dependencies

## Bun

You need to have [bun](https://bun.com/) installed on your machine.

Run `bun install` to install the necessary dependencies.

# Running tests

To run the test suite, simply run `bun test`.

# Code hygiene

We use various coding conventions throughout the project which we expect
contributors to follow to the best of their ability. Most of them come from [ultracite](https://docs.ultracite.ai/) with [biome](https://biomejs.dev/).

Run `bun fix` to fix and check for linting and formatting errors.

# Commit hygiene

We care about our commit history, both for historic purposes and to aid with reviewing. Try to follow [conventional commits](https://www.conventionalcommits.org/en/v1.0.0/).
